# Algorithms

Java implementations of core algorithms from coursework at Loyola University Chicago (COMP 363 — Design and Analysis of Algorithms).

## Contents

| Directory | Algorithm |
|---|---|
| `astar` | A* search |
| `changeMaking` | Change-making (dynamic programming) |
| `dijkstra` | Dijkstra's shortest path |
| `divideConquer` | Divide and conquer problems |
| `graph` | Graph traversal (BFS/DFS) |
| `huffman` | Huffman encoding |
| `mst` | Minimum spanning tree (Kruskal / Prim) |
| `sorting` | Sorting implementations |
| `sorting.algorithms` | Sorting algorithm variants |
| `sortingcomplexity` | Complexity analysis comparisons |
